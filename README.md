# Bike Shop Sales Dashboard

## Overview
This project is a Power BI dashboard built to analyze sales performance for a bike shop. The dashboard provides insights into revenue, profit, customer behavior, product performance, and sales trends.

## Objectives
- Analyze overall sales and revenue performance.
- Track profit and business growth over time.
- Identify best-selling products and product categories.
- Understand customer purchasing behavior.
- Support data-driven business decision making.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Visualization
- Data Cleaning

## Key Insights
- Sales performance can be monitored by time period, product category, and customer segment.
- The dashboard helps identify high-performing products and revenue trends.
- Interactive filters allow users to explore business performance from different perspectives.
